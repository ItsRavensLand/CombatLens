CombatLens

CombatLens is a Minecraft PvP analytics plugin for Paper servers.

The goal of this project is to provide detailed combat statistics, fight history tracking, and post-fight analysis for players and server administrators.

Current Status

⚠️ This plugin is currently under active development.

This repository serves as a development log and source code backup while the plugin is being built. Features may change, be incomplete, or contain bugs until the first stable release.

Planned Features

- Fight history tracking
- Detailed combat statistics
- Hit and combo analysis
- Damage breakdown
- Healing and consumable tracking
- Projectile statistics
- Equipment analysis
- GUI-based fight viewer
- Additional combat insights and analytics

Development Progress

Updates are pushed regularly as development continues.

The project is not considered production-ready yet.

Target Platform

- Paper 1.21.x
- Java 21+
